BrowserStack Automate API Client v2

18 October 2016 -

I’ve just released version 2.0.1 of the BrowserStack Automate .NET client open source project that I maintain on GitHub.

READMORE

This release adds functionality for consuming more of the BrowserStack Automate API, such as for deleting resources, but also adds support for targeting .NET Core via the use of .NET Standard 1.3.

You can download the latest version of the client from NuGet or fork it on GitHub.
